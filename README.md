# AI Marketing Agent

An AI-powered marketing agent for campaign evaluation, decision support, scenario simulation, and dashboard-based analysis.

## Overview

This project analyzes marketing campaign data and generates structured recommendations based on performance, efficiency, maturity, and trend signals. It also includes a dashboard for reviewing results and exploring campaign outputs.

## Main Features

- campaign evaluation with rule-based goals
- decision output with stance, severity, and reasons
- scenario simulation for budget changes
- dashboard for inspecting outputs
- optional LLM-based advisor output
- memory and run storage for historical context
